1) Что делает fetch?

- fetch - это встроенная функция JavaScript, которая позволяет отправлять HTTP-запросы и получать данные с сервера, используется для работы с API, загрузки данных, отправки форм и многого другого.

2) Зачем нужны async/await?

- async/await это синтаксис JavaScript, который помогает писать ассинхронный код как синхронный, не заменяя промисы. 
- async засьавляет функцию всегда возвращать промисы.
- await ставит выполнение фкнцкции на паузу, пока промис не выполнится, затем разварачивает его и кладет результат в переменную.

3) Как работает localStorage?

- localStorage - это маленькая база данных прямо в браузере, которая решает проблему "умирания" JS переменных при перезагрузке страницы.
